# Among The Genres
*Among The Genres* is an Administrative side of Online BooksStore with diffrent genres of books.


## Tools and Technologies ##
* EJS
* Express
* NodeJS
* MongoDB
* Admin LTE

## Web Pages ##
Port:http://localhost:1000/home
